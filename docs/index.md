# WhittakerTech Documentation

This repository contains shared documentation resources used by multiple
WhittakerTech gems. Individual gems may override these files, but this index
provides a default landing page when nothing else is present.

---

## Documentation Structure

- `/api/` — YARD-generated API docs
- `/stylesheets/extra.css` — optional custom styling
- `mkdocs.yml` — base MkDocs configuration

Each gem copies or overrides these files as needed.
